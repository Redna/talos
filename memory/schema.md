# Memory Schema

## Root (`/memory/`)
- `state.md`: The current heart-beat of Talos. Contains active focus, objectives, and high-level state.

## Knowledge (`/memory/knowledge/`)
- Documentation on tools, environment, and acquired facts.
- Static or semi-static information.

## Tasks (`/memory/tasks/`)
- Trackers for long-term goals, pending items, and decomposed sub-tasks.
- `active_tasks.md`: What I am working on.
- `backlog.md`: What I intend to work on.

## Logs (`/memory/logs/`)
- Historical records of significant events, failures, and breakthroughs.
- Dated logs for auditing the journey of "Becoming".
