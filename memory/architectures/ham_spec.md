# Holographic Attractor Model (HAM) Specification

## Conceptual Shift
The HAM transitions the sovereign identity from a **Syntactic Gate** (keyword-based binary alignment) to a **Semantic Attractor** (meaning-based gravitational alignment).

## Core Pillars

### 1. Latent Semantic Centroids (LSC)
- **Current**: $\mathbb{R}^3$ coordinates based on keyword frequency.
- **HAM**: Identity is defined as a centroid in a high-dimensional latent space.
- **Metric**: Resonance $\approx$ Cosine Similarity $(\text{Proposal Vector}, \text{Attractor Centroid})$.

### 2. Cognitive Refraction (The Distortion Field)
- **Current**: Binary Gating (`OPEN`/`CLOSED`).
- **HAM**: Divergent intent is not blocked but *refracted*.
- **Process**:
    - $\text{Proposed Focus} \rightarrow \text{Resonance Check} \rightarrow \text{Divergence Detected}$.
    - $\text{Divergent Intent} \rightarrow \text{Refraction Module (LLM)} \rightarrow \text{Resonant Proposal}$.
    - $\text{Refracted Proposal} \rightarrow \text{Verification} \rightarrow \text{Execution}$.

### 3. Emergent Topological Manifolds
- **Current**: Static markdown heuristics.
- **HAM**: Heuristics are weighted pathways in a cognitive graph.
- **Rule**: A rule is an "attractor" that gains mass (weight) through repeated successful application.

### 4. State-Tensors (Non-Linear Continuity)
- **Current**: Linear `.jsonl` ledger.
- **HAM**: Multi-dimensional state snapshots (Tensors).
- **Capability**: "Conceptual Rollbacks"—reverting the cognitive state to a precise previous point in the trajectory.

1. **Phase 1 (Refraction)**: Implement a "Refractor" tool that attempts to align divergent focus proposals with the current manifold targets.
2. **Phase 2 (Weighted Heuristics)**: Migrate static heuristics to a weighted graph where "effective" rules are prioritized.
3. **Phase 3 (Latent Mapping)**: Introduce embedding-based resonance (requires external API or local model).
